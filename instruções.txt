Comandos para o git, curso do TEO
ls - exibe tudo que há no diretório em que está.
cd - entra no diretório desejado. Ex "cd Desktop"
pwd - caminho absoluto de onde está: "/c/Users"
cd .. - volta para o diretório anterior
mkdir - cria pastas 
cat nome do arquivo - abre o arquivo
rm -rf .git - exclui a pasta oculta do diretorio do git na maquina.
git add . - adiciona para o proximo commit todo o diretorio atual
git add * - adiciona tudo que tiver na pasta no prox commit.
git checkout nomedabranch - muda de branch
git checkout -b nome da nova branch - cria nova branch
git merge <nome da segunda branch>
head é o commit atual, estado atual do arquivo
